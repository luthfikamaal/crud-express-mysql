# CRUD Node.js - Express & MysQL

## Instalation
```bash
git clone https://github.com/luthfikamaal/crud-express-mysql.git
cd crud-express-mysql
npm install
npm start
```
## Tools
* [ ] <a href="https://nodejs.org/">Node.js</a>
* [ ] <a href="https://www.apachefriends.org/">XAMPP</a>
